🔑 Update Credentials

rclone credentials for Proton Drive (username, password, 2FA) are stored encrypted in ~/.config/rclone/rclone.conf. To update them, run the following in a terminal: 

rclone config reconnect protondrive:

This will prompt for your Proton credentials interactively and update the stored config. To add a new remote entirely, run rclone config.
